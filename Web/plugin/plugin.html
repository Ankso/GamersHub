The plugin installation is really easy.
  • The plugin download should start in a few seconds. If not, click this link.
  • Execute the downloaded file, it will automatically install the plugin.
  • After the plugin installation, restart your web browser.
And that's it! You can now enjoy all the site features.

NOTE: This plugin accesses the running processes list in your system to find possible games that you are playing. But this data is not collected in any way by us. The only thing that we care about is what you are playing.
